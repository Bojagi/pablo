# ButtonBar

A layouting component to put multiple buttons next to each other aligned to the right. 
It's handy for dialogs and confirmations.

## Example

<Playground>
  <Card width={500}>
    <Title>Do you want to proceed?</Title>
    <Paragraph>Please select one of the buttons below!</Paragraph>
    <ButtonBar>
      <Button size="medium" variant="text" color="plain">
        Cancel
      </Button>
      <Button size="medium">Accept</Button>
    </ButtonBar>
  </Card>
</Playground>
