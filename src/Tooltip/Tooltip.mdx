# Tooltip

A component that decorates it's child with a tooltip that shows hints/information text on hovering over the child.

## Examples

### Plain tooltip

<Playground>
  <Tooltip content="A tooltip shows on hovering (you just witnessed)">
    <Button color="plain">
      Hover over me
    </Button>
  </Tooltip>
</Playground>

### Tooltip positioning

A tooltip can be positioned on each of the for sides of it's child. 

<Playground>
  <Box display="flex" mx={-4}>
    <Tooltip side="left" content="Left tooltip">
      <Button mx={1} color="plain">
        Left
      </Button>
    </Tooltip>
    <Tooltip side="top" content="Top tooltip">
      <Button mx={1} color="plain">
        top
      </Button>
    </Tooltip>
    <Tooltip side="bottom" content="Bottom tooltip">
      <Button mx={1} color="plain">
        bottom
      </Button>
    </Tooltip>
    <Tooltip side="right" content="Right tooltip">
      <Button mx={1} color="plain">
        right
      </Button>
    </Tooltip>
  </Box>
</Playground>

### Tooltip by click

By default the tooltip shows on hover. If you want to show the tooltip on click, you can use the `showOnClick` boolean prop.
This disables the hover behavior and shows the tooltip on click.

<Playground>
  <Tooltip content="You only get me by click" showOnClick>
      <Button color="plain">
        Click me
      </Button>
    </Tooltip>
</Playground>

### Tooltips with a delay

If you want to only show a tooltip after hovering over it for a certain time, you can use the `delay` prop and pass
a number that represents a millisecond time. This is useful for tooltips for more obvious actions.

<Playground>
  <Tooltip content="Sorry for being late" delay={500}>
      <Button color="plain">
        Quickly, hover over me!
      </Button>
    </Tooltip>
</Playground>

### Usage on disabled children

When a child has the `disabled` prop set to true, the tooltip doesn't show as well. If you want to get around this
behavior you need to wrap the child in a `div`, `span` or any other element. 

<Playground>
  <Tooltip content="I'm not there!">
    <Button mx={1.5} color="plain" disabled={true}>
      I am not clickable
    </Button>
  </Tooltip>
</Playground>

### Using animation

By default the tooltip has a slide in animation with a `100` millisecond duration. You can change this by using the following props:

* `animation` - a pablo animation component (default `SlideAnimation`)
* `animationDuration` - The duration of the animation in milliseconds (default `100`)
* `animationReverse` - If the animation should be reversed (default `false`)

<Playground>
  <Tooltip content="I am really slow" animationDuration={5000}>
    <Button mx={1.5} color="plain">
      Quickly, hover over me!
    </Button>
  </Tooltip>
</Playground>
